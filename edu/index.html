1-x 認識日間手術 規劃中 · 預計 2 份

病人最常問的「為什麼可以當天回家?」、「我適合做日間手術嗎?」— 定錨整套衛教的論述基調。

規劃中 · 預計 2 份
  • 1-1什麼是日間手術 · 為什麼可以 < 24 小時回家
  • 1-2適合 / 不適合日間手術的條件
2-x 三大手術族群介紹 規劃中 · 預計 3 份

病人快速找到自己的手術 — 上肢骨科、椎體成形、膀胱鏡。每篇聚焦該族群的術式、麻醉、止痛、特殊注意事項。

規劃中 · 預計 3 份
  • 2-1上肢骨科(肩關節鏡 / 腕隧道 / 扳機指 / 橈骨遠端骨折)
  • 2-2椎體成形術(vertebroplasty)
  • 2-3膀胱鏡 / 輸尿管鏡(TURBT / URS)
3-x 麻醉方式 規劃中 · 預計 4 份

日間 ERAS 麻醉主軸:區域阻斷為主、短效全麻為輔。讓病人理解為什麼這樣選、會有什麼感受、清醒後感覺怎麼回來。

規劃中 · 預計 4 份
  • 3-1全身麻醉(短效藥物為主)
  • 3-2區域阻斷(肩部 / 上肢神經阻斷)
  • 3-3脊椎 / 硬膜外麻醉
  • 3-4監測麻醉 MAC(輕鎮靜 + 區域)
4-x 術前準備 規劃中 · 預計 3 份

日間 ERAS 核心差異化點 — 清流質可至術前 2 小時 + 碳水化合物負荷(不是傳統 8 小時禁食)、停藥指引、當天注意事項。

規劃中 · 預計 3 份
  • 4-1術前禁食指引(清流質 2 hr + 碳水化合物負荷)
  • 4-2術前停藥指引(抗凝血 / 抗血小板 / 降血糖)
  • 4-3當天注意事項(服裝 / 陪同人 / 貴重物品)
5-x 術後核心 規劃中 · 預計 4 份

日間 ERAS 多模式策略 — 非鴉片止痛、多模式止吐、區域阻斷感覺回來、早期口服與下床。讓病人理解每一個藥物與動作的意義。

規劃中 · 預計 4 份
  • 5-1多模式止痛(非鴉片為主)
  • 5-2PONV 預防與處理(術後噁心嘔吐)
  • 5-3區域阻斷的「感覺漸漸回來」— 12-24 hr
  • 5-4早期口服 + 早期下床
6-x 離院標準與流程 日間獨有 規劃中 · 預計 2 份

PADSS 五指標 — 日間手術 ERAS 最核心的離院關卡。為什麼要等到分數達標,而不是「醒了就走」。

規劃中 · 預計 2 份
  • 6-1PADSS 五指標 — 為什麼要等到達標才能走
  • 6-2出院前確認單 + 24/7 緊急聯絡
7-x 回家後 24-72 小時 規劃中 · 預計 3 份

回家不等於結束 — 神經阻斷退去後的疼痛、夜間警訊、給陪伴者的觀察重點、24 / 48 / 72 小時 follow-up 流程。

規劃中 · 預計 3 份
  • 7-124 小時警訊清單 + 何時聯絡
  • 7-2給陪伴者的衛教(夜間觀察重點)
  • 7-3follow-up 流程(LINE / 電話追蹤)
關於這份目錄 — 本目錄為日間麻醉加速康復團隊內部維護的衛教資源總覽。各單張依病人個別情況(手術族群 / 共病 / Apfel 嘔吐風險 / 麻醉方式),由醫療團隊透過 LINE / Email / 紙本等方式於合適時機提供給病人與家屬。如需擴充或調整單張內容,請聯繫日間麻醉加速康復團隊。

每一份衛教,都是為了讓病人在 24 小時內安心回家而設計。理解 → 預備 → 主動參與 — 這是本院日間麻醉 ERAS 加速康復的核心精神。
